# NDF Examples

This folder contains example Note Data Format (`.notedf`) files demonstrating various syntax features.

## Files

- `config.notedf` - Comprehensive example showing all NDF features
- `FORMAT_COMPARISON.md` - Detailed comparison of NDF, JSON, and YAML formats

## Features Demonstrated

- Simple key-value pairs
- Lists (space and comma separated)
- Nested objects with indentation
- Arrays
- Inline objects
- Multi-line text (using `|` syntax)
- References (variables starting with `$`)
- Type hints (`@time`, `@embedding`, `@f[n]`)
- Null values (`null`, `none`, `-`)
- Boolean values (`yes`, `no`, `true`, `false`)
- Comments (both line and inline)

Open these files in VS Code with the NDF extension installed to see syntax highlighting in action.
